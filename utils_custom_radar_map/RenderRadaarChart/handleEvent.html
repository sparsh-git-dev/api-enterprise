Feeds our gesture recognizers for handling events, we also handle PointerHoverEvent here.

Our gesture recognizers are responsible for notifying us about happened gestures (such as tap, panMove, ...) we need to give them PointerDownEvent then they will listen to the global GestureBinding for further events.

We need to handle PointerHoverEvent because there is no gesture recognizer for mouse hover events (in fact they don't have any gestures, they are just events).

Implementation

@override
void handleEvent(PointerEvent event, covariant BoxHitTestEntry entry) {
  assert(debugHandleEvent(event, entry));
  if (_touchCallback == null) {
    return;
  }
  if (event is PointerDownEvent) {
    _longPressGestureRecognizer.addPointer(event);
    _tapGestureRecognizer.addPointer(event);
    _panGestureRecognizer.addPointer(event);
  } else if (event is PointerHoverEvent) {
    _notifyTouchEvent(FlPointerHoverEvent(event));
  }
}
