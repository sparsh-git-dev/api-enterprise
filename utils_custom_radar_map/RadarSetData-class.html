RadarSetData class

the data values for drawing RadarChart sections

RadarSetData({List<RadarEntry>? dataEntries, Color? fillColor, Color? borderColor, double? borderWidth, double? entryRadius})
RadarChart can contain multiple RadarSetData And it shows them on top of each other. each RadarSetData has a set of dataEntries and the RadarChart uses this dataEntries to draw the chart.
